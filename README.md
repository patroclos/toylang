## JayJay - the new big thing ##

JayJay is a simple compiled programming language developed for the lulz

## Syntax ##

1. Function definition
   ```
   TYPE name(ARGS) { CODE }
   ```
   Example:
   ```
   int test()
   {
    ...
   }
   ```

2. Return statements
    ```
    return EXPRESSION
    ```
    Example:
    ```
    return 2 + 5
    ```
3. Types
    - int : 64 bit integer
    - double : double (obviously)
